# Academia del CIO

Plataforma interna para gestionar el ciclo completo de entrevistas a líderes
tecnológicos (CIO/CTO/CEO): desde la identificación de un candidato hasta la
relación a largo plazo, pasando por contacto, agendado, grabación y contenido.

> **La relación con el entrevistado importa más que el contenido.**
> El contenido es una consecuencia de la relación.

## Estado actual

**Fase 0 — Cimientos** (sistema de trabajo del proyecto). Todavía **no** hay
código de aplicación: esta fase monta el contexto, la documentación, las reglas
y las herramientas con las que construiremos el producto.

## Cómo está organizado el conocimiento

- **[CLAUDE.md](./CLAUDE.md)** — manual operativo mínimo (reglas y comandos siempre activos).
